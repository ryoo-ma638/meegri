# meegri 〜ミーグリ練習シミュレーター〜

> アイドルの「ミーグリ（オンライン・ミート＆グリート）」を、本番そっくりに練習できるブラウザ完結型のWebアプリ。

**🎮 デモ（ブラウザですぐ遊べます）→ https://meegri.vercel.app**

持ち時間・通信ラグ・時間切れの緊張感、そして生成AIが演じるアイドルの「機嫌」や「反応」までリアルに再現。会話のあとはAIがあなたの“話しかけ方”を採点します。インストール不要・無料、スマホ／PC対応。

> ※登場するのは実在の人物・団体・サービスとは無関係の **架空オリジナルキャラクター** です。

---

## ✨ 主な機能

**会話・AI**
- 生成AI（Gemini）が演じるアイドルと、制限時間つきで1対1のビデオ通話風トーク
- 会話全体をAIが講評・採点（スコア／ランク／良かった点／助言／一言）
- AIが使えない時は内蔵の「台本＋ルール」エンジンへ自動フォールバック（鍵なしでも遊べる）
- 音声入力（マイクで話す）＆読み上げ（TTS）

**“本番らしさ”の演出**
- 公平なタイマー：通信ラグは持ち時間に含め、AIの思考時間は含めない
- 機嫌システム：繰り返し・失礼・一方的・無言などに、表情と塩対応で反応
- 好感度＋推し度（レベル・称号・ハート）／一途は加点・他の子の話はやきもち
- 名前ごとに別々の関係・記憶（前回の続きを覚える会話メモリ）
- 本番リハーサルモード（通話テスト→枚数確認→本人確認→待機列…の準備フロー）
- 参加枚数（1〜20）・部（1〜5部）・難易度（持ち時間／ラグ／無言をスライダー調整）
- エンディング図鑑 全23種（終わり方で変化・コンプリート率つき）

**ビジュアル**
- 背景：WebGL / GLSL シェーダーによるオーロラ演出
- アイドル：インラインSVGの立ち絵（まばたき・口パク・呼吸／3人を髪型・瞳・衣装で差別化）
- カメラ：自分の顔を WebGL ビューティー加工で表示（**端末内のみ・録画/送信なし**）

**運用**
- AI会話のレート制限（1日5回・超過で台本モードへ自動切替）＋オーナーモード（無制限）
- 記録・記憶は localStorage に保存（端末内）。個別削除・全リセット・「記録を残さない」練習モードあり

## 🛠 技術構成

- **フロント**：単一HTML ＋ バニラ JavaScript（**フレームワーク・ビルド・npm依存なし**）
- **グラフィックス**：WebGL / GLSL（背景シェーダー・カメラのリアルタイム加工）、インラインSVG
- **音声**：Web Speech API（音声認識・音声合成）、Web Audio（効果音生成）
- **AI**：Google Gemini（`gemini-2.5-flash`）
  - ローカルは依存ゼロの `server.js`、公開は Vercel サーバーレス関数 `api/reply.js`
  - **APIキー・オーナーコードはサーバー側の環境変数のみ**（クライアントJSには一切出さない）
  - 応答は responseSchema で JSON 強制、safetySettings で誤ブロック対策、失敗時は台本へフォールバック
- **永続化**：localStorage（図鑑・自己ベスト・会話メモリ・関係性）

> 最新の Chrome 系ブラウザ推奨（WebGL・最新CSS・getUserMedia を使用）。

## 🎮 動かし方（3通り）

### A. いちばん簡単（AIなし・台本モード）
`app/index.html` を **ダブルクリック** するだけ。
※カメラ・音声・AI会話を使うには、下記 B / C（セキュアな文脈）が必要です。

### B. ローカルでAI会話つき（おすすめ）
1. Gemini APIキーを無料取得 → https://aistudio.google.com/apikey
2. キーを設定（**自分だけが扱う**。誰にも渡さない）:
   ```sh
   cp .env.example .env
   # .env を開いて GEMINI_API_KEY=取得したキー を記入
   ```
3. 起動（Node 18+・インストール不要）:
   ```sh
   node server.js
   ```
4. ブラウザで **http://localhost:8753** を開く（カメラ許可でビューティーも有効）

### C. ネットに公開（Vercel）
1. リポジトリを Vercel にインポート
2. **Settings › Environment Variables** に登録（**コードには書かない**）:
   - `GEMINI_API_KEY` … 必須（AI会話・AI採点用）
   - `OWNER_CODE` … 任意（オーナーモード用）
   - `GEMINI_MODEL` … 任意（既定 `gemini-2.5-flash`）
3. Deploy（`vercel.json` によりビルドなしで `app/` を静的配信 ＋ `api/` を関数として実行）

> 静的ホスティング（GitHub Pages 等）でも、台本モードでフルプレイ可能です。

## 🔑 APIキー・オーナーコードの扱い（重要）

- キーとオーナーコードは **サーバー側（`.env` / Vercelの環境変数）だけ** に置きます。**フロントのHTML/JSには一切書きません**。
- `.env` は `.gitignore` 済み＝GitHubには上がりません（共有用の `.env.example` のみ同梱）。
- 万一キーを公開してしまった場合は、すぐ Google AI Studio で **再発行（無効化）** してください。

## 📁 構成

```
app/index.html   フロント（アプリ本体・単一HTML）
server.js        ローカル用サーバー（静的配信 + /api/reply・依存ゼロ）
api/reply.js     公開(Vercel)用サーバーレス関数（AI会話・AI採点・オーナー照合）
vercel.json      Vercel設定（ビルドなし・app/静的配信・api/関数）
.env.example     環境変数のテンプレ（.env は各自作成・非公開）
ROADMAP.md       実装済み機能・今後の予定
```

## 🎭 キャラクター・著作権について

- 登場するアイドル（夏目ゆい／星川ひなの／花村こはる）はすべて **架空のオリジナルキャラクター** で、実在の人物・団体・サービスとは一切関係ありません。
